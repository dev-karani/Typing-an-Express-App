# Typing an Express App

A minimal typed Express.js application that returns a list of pets.

## Overview
This is a typed REST API built with Express.js that serves pet data in JSON format.

## Tech Stack
- Typescript
- Node.js
- Express.js

## Endpoints
- `GET /pets/` — returns available pets
- `GET /pets/:id/?password=please` — returns specific pet
- `GET /pets/?adopted=true&species=dog"` — returns filtered as per parameters
